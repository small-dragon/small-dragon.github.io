Gulp实战 加强版，前端需要的功能基本上都用了，需结合 Gulp实战篇一起观看

这个脚手架有什么用呢

首先，还是分本地版src和发布版dist来说吧

本地版

  1. 开启静态服务器、设置代理服务器，监听less、html、js文件内容实现热加载
  2. 实现类似于 java 后端的 include 功能
  3. less插件的使用、less文件转化为一个css文件，并注入 sourcemaps到css文件中
  4. 合并本地js文件到一个js文件（如main.js）

发布版

  1. CSS文件和JS文件进行MD5处理
  2. CSS文件进行压缩处理，大小减少一半多,并且没有注入sourcemaps
  3. JS文件进行压缩处理
  4. 根据dist需要的其他静态资源（如images文件夹中的图片、sound文件夹中的音频）从src转移过来

怎么用

  • 首先，你得先建个类似于下图的文件夹
  • 用 gulp default或 gulp 就可以使用本地版咯
  • 最后，用 gulp dist 就可以生成发布版dist文件夹咯，压缩一下就可以生成dist.rar发给部署人员去部署到服务器啦，是不是很方便
